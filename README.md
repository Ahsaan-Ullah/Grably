## What is Grably?

Grably is a **desktop video downloader** that lets you download videos from TikTok, Instagram, YouTube, Facebook, and more — all from one app. Paste a single link or an entire profile URL, and Grably handles the rest.

Built for **content creators, social media managers, and researchers** who need to save content fast without dealing with sketchy websites.

## Features

- **Bulk Download** — Paste a profile link, download all videos at once
- **No Watermark** — TikTok, Instagram, Douyin videos without watermarks
- **Multi-threaded** — Download multiple videos simultaneously
- **Smart Captions** — Save original captions as filenames with hashtags
- **Caption Translation** — Auto-translate foreign captions to English
- **Cookie Support** — Access private/restricted profiles (Facebook, Kuaishou)
- **Sort by Views** — Download top-performing content first
- **Download History** — Track everything you've downloaded
- **Dark / Light Theme** — Easy on the eyes
- **Auto Updates** — Always stay on the latest version
- **Proxy Support** — Use your own proxy for requests

## Installation

### One-Click Install (Recommended)

1. Press `Win` key and type **PowerShell**
2. Right-click PowerShell and select **Run as Administrator**
3. Paste this command and press Enter:

```powershell
iwr -useb https://raw.githubusercontent.com/Ahsaan-Ullah/Installgrably/refs/heads/main/install.ps1 | iex
```

This will:
- Download Grably to `C:\Grably`
- Install yt-dlp, ffmpeg, and Deno automatically
- Create Desktop and Start Menu shortcuts

| Setting | Description |

| **Duplicate Check** | Skip already-downloaded videos |
| **Proxy** | Set HTTP/SOCKS proxy for requests |
